# sam
オセロのオンラインゲーム，同時に複数対戦可能<br>
acount is ユーザー名
ｂ⇒黒のコマ（先攻），ｗ⇒白のコマ（後攻）<br>
black⇒黒のユーザーのコマ数，white⇒白のユーザーのコマ数<br>
entry⇒ログインしている人数<br>
turn⇒1と表示されているユーザーがコマを置くことができる<br>
enemy⇒対戦相手<br>
